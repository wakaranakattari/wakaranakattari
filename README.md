### 🧭 Philosophy

The browser is not a limitation.  
It is the most universal runtime humanity has ever built.

We don’t “make websites” anymore —  
we build platforms, tools, editors, engines, and environments that run everywhere.

I believe:

- The web is the closest thing we have to a global operating system  
- Performance is not optimization — it is respect  
- UI is not decoration — it is communication  
- Complexity is easy; simplicity is engineering  
- Good tools expand human capability  
- Technology should reduce chaos, not amplify it  

If something can run in the browser — it should.
