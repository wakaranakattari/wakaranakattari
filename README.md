## About me
> I'm a software engineer who loves exotic and strange languages.
>
> I started writing in these languages when I grew tired of conventional
> programming, of ordinary languages that differ little from one another.
> These tools became a lifesaver for me, preventing professional burnout.
>
> I love promoting these tools to my small community so that people can
> take a break from their daily work and appreciate the elegance of these
> solutions.
